Scattered Polaroids Image Gallery is free gallery plugin with CSS3 3D transforms which scatters the images as polaroids throughout the given section.

== Description ==

Scattered Polaroids Image Gallery is free gallery plugin with css3 3D transforms which scatters the images as polaroids throughout the given section and brings them forward one by one. Just use the shortcode [SPIGallery] and Voila! you get a Scattered Polaroids Image Gallery. Now just add the images from the "Add images" menu, and you're Done! Very Easy and Simple to use! Download, Use and Enjoy! A bunch of thanks in advance! 

Benefits that you will get from this plugin are:-
1. Built-in TinyMCE button provides full scope to enter the shortcode in any rich editor. 
2. Built with CSS3 for minimum loading time and better performance
3. Built with a responsive layout for coverage of all devices
4. Maximum customizability 
5. Ability to add an unlimited number of images 


== Installation ==

This section describes how to install the plugin and get it working.


1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Scattered Polaroids Image Gallery screen to configure the plugin
4. Use the Add Images screen to add your images.
5. Add the shortcode [SPIGallery] in any post, page or editor manually or use the button provided for the same. 
6. You're Done!!!


== Frequently Asked Questions ==
